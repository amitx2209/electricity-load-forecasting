\# Electricity Load Forecasting using Machine Learning



This project predicts future electricity demand for a selected Indian state

using historical consumption data and multiple machine learning models.



\## Models Used

\- Linear Regression

\- Random Forest Regressor

\- Gradient Boosting Regressor

\- Support Vector Regressor



\## Features

\- Lag features

\- Rolling mean

\- Calendar features



\## Tools

\- Python

\- Pandas, NumPy

\- Scikit-learn

\- Streamlit

\- Jupyter Notebook
